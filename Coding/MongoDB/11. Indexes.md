### Query Explanation
```
db.students.find({name: "Larry"}).explain("executionStats")
```
### Creating Index
```
db.students.createIndex({name: 1})
```
The `name` is a field name.
### Checking Index Name
```
db.students.getIndexes()
```
### Dropping Index
```
db.students.dropIndex("name_1")
```
`name_1` is the name of index.
