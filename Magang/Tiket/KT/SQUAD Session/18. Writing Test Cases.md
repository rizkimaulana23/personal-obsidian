## Test Case Writing
A test case is a set of actions executed to verify a particular feature or functionality of your software application. 

- Test Condition
- Test Steps
- Test Data
- Precondition
- Expected result
- Status
- Priority 
- Environment
- Platform
- Type of Test Case (Positive, Negative, Performance)
- Automated (Yes/No/Can not automate)
